# Car Sales Analytics Dashboard (Power BI)

## 📊 Overview
This project focuses on analyzing car sales data using Power BI to generate meaningful business insights.

## 🚀 Key Features
- Interactive dashboards with slicers and filters  
- KPI tracking (YTD Sales, Avg Price, Cars Sold)  
- Region-wise and category-wise sales analysis  
- Drill-down insights for better decision-making  

## 🛠 Tools Used
- Power BI  
- DAX  
- Power Query  

## 📈 Insights
- Identified sales trends over time  
- Compared performance across regions and categories  
- Highlighted key metrics for business monitoring
